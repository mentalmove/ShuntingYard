## To compile without IDE

```
mkdir build
javac -sourcepath . -d build yard/*.java
cd build
```

Create file `manifest.txt` with content
```
Main-Class: yard.Yard

```
(New line in the end is mandatory)

```
jar cfm ShuntingYard.jar manifest.txt *
```


## Run

```
java -jar ShuntingYard.jar
```
OR
```
java -jar ShuntingYard.jar '2 + 3 * 4 or whatever'
```
